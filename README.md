# 🧮 Random Math Game

A simple and interactive math quiz game built using vanilla JavaScript.

## 🚀 Live Demo

👉 [Click here to play](https://mathgamejs12.netlify.app/)

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript

## 🎮 Features

- Randomly generated math questions
- Score tracking based on correct answers
- Countdown timer
- Responsive UI
